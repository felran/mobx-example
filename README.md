## mobx example介绍

基于mobx在react中应用的一个脚手架工程，支持ES6和JSX语法，使用webpack4+koa+react+mobx 实现


* 使用mobx装饰器语法
* 使用koa框架结合webpack中间件技术
* 支持热更新React-Hot-Loader
* 打包支持webpack dllPlugin，加快打包速度

### 运行这个Demo

```
npm install 
npm run dev
```

打开浏览器访问 [localhost:3000](http://localhost:3000)


### 打包
```
npm run dll 
npm run build
```
